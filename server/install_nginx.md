#Nginx安装
````
yum install nginx
````
> Nginx命令
. 'whereis nginx' 查看nginx的默认路径
. 'nginx -v'      查看nginx版本
. 'nginx -v'      查看nginx版本
> Nginx重启
````
nginx -c /etc/nginx/nginx.conf
nginx -s reload
````
